## ✨ Introduction
* 국가에서 지원해주는 청년 일자리 사업 구인 정보 모음 앱
* 지역마다 사이트가 흩어져 있고 그 마저도 사업장 모집 글과 청년 모집 글이 분리되어 있지 않아서 구직자 입장에서 보기 불편했던 기존의 청년일자리 사업 공고방식을 개선
* 국가 고용 지원 사업의 구인 공고를 한눈에 볼 수 있고 지역, 제목 별 검색이 가능해 필요한 정보를 빠르게 확인할 수 있다
</br>
